# Bakkhali-Weather-Dataset-Hourly-Multi-Year-
This dataset contains multi-year hourly meteorological data for Bakkhali, a coastal region in West Bengal, India (Lat: 21.56, Lon: 88.26). The data has been obtained from the NASA POWER (Prediction Of Worldwide Energy Resources) API, a research-grade satellite and climate reanalysis dataset widely used for renewable energy studies.
